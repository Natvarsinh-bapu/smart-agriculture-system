# Smart Agriculture System 🌱

A complete IoT-based Smart Agriculture project that combines:

- 🌾 Real-time sensor monitoring
- 🤖 AI-based crop recommendation
- 💧 Smart irrigation monitoring
- 📊 Live web dashboard visualization

This repository contains all source code related to the project including:

- Wokwi simulation code
- Google Colab machine learning code
- Next.js dashboard frontend
